Κάνεις τα εξής βήματα:
1. Συνδέεις το raspberry με ενα καλώδιο USB στο λαπτοπ, ανοίγεις ενα παράθυρο terminal και εκτελείς την εντολη "iwconfig". Θα σου εμφανίσει κάτι ονόματα αλλά εσύ θα ψάξεις αυτό που είναι κάπως έτσι "enpXsXXuX" (όπου Χ αριθμός) και το σημειώνεις κάπου.
2. Ανοίγεις ένα παράθυρο terminal και εκτελείς την ακόλουθη εντολή: "sudo nano /etc/network/interfaces" και θα σου εμφανίσει ένα παράθυρο που θα γράφει κάτι τέτοιο:
# interfaces(5) file used by ifup(8) and ifdown(8)
auto lo
iface lo inet loopback
3. Σε αυτό το παράθυρο θα γράψεις κάτω από αυτό που θα δεις το εξής:
iface enpXsXXuX inet static
           address 192.168.0.[2-254]
           netmask 255.0.0.0
           gateway 192.168.0.1
όπου:
address: η στατική διεύθυνση του λάπτοπ που πρέπει να είναι στο εύρος 192.168.0.[2-254]
netmask: το βάζεις όπως το έχω και δεν το αλλάζεις
         gateway: η διεύθυνση του Raspberry. Για αρχή βάλτη όπως την έχω (είναι η default που έχει), αργότερα όμως που θα την αλλάξεις μην ξεχάσεις να την αλλάξεις και εδώ.
4. Μετά ανοίγεις τις ρυθμίσεις και ψάχνεις την καρτέλα που λέει "Network". Αφού έχεις συνδέσει το Raspberry θα πρέπει να σου έχει μια επιλογή που λέει "USB Ethernet". Ανοίγεις τις ρυθμίσεις, πας στην καρτέλα που λέει "IPv4", επιλέγεις "Manual" και εισάγεις στα αντίστοιχα πλαίσια τις διευθύνσεις που όρισες στο προηγούμενο βήμα. Μόλις σου δείξει "Connected", κάνεις SSH και είσαι έτοιμος.
Το τελευταίο βήμα πρέπει να το κάνεις κάθε φορά που θα αποσυνδέεις το Raspberry και θα το ξανασυνδέεις αργότερα
